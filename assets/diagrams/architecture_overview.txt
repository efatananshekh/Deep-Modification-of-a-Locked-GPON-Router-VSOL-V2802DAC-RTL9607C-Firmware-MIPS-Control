# ======================== HARDWARE QOS REGISTERS =============================

Hardware QoS via Realtek RTL9607C:
-----------------------------------

         /proc/rg/
         ├── assign_access_ip      = 0xc0a80124 (192.168.1.36 = Gaming PC)
         ├── qos_ring_0            = 1 (Priority ring for 192.168.1.36)
         ├── hwnat                 = 1 (Hardware NAT enabled)
         ├── shortcut              = 1 (L2/L3 shortcut enabled)
         └── rg_api                (RTK RG API interface)

    Priority Bands (prio_queue):
    ┌─────────────────────────────────────────────────────────────┐
    │  Band 0 (Highest) │ Gaming/VoIP │ TOS=0x10 (Minimize-Delay) │
    │  Band 1           │ Normal      │ TOS=0x00                  │
    │  Band 2           │ Bulk        │ TOS=0x08 (Maximize-Thru)  │
    │  Band 3 (Lowest)  │ Background  │ TOS=0x02 (Minimize-Cost)  │
    └─────────────────────────────────────────────────────────────┘
